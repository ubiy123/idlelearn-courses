# L1-01 项目作业：我的投资世界观初稿 + 财务自检表

## 作业目标

通过写作和计算，把本课理论转化为个人化的投资认知和行动起点。

## 提交格式

提交一份 Markdown 文件，文件名：`investment-worldview-L1-01.md`。

## 内容要求

### 一、投资世界观初稿（300-500 字）

请围绕以下问题展开：

1. **你为什么想学习投资？**
   - 是被动原因（钱在贬值）还是主动原因（想实现某个目标）？
   
2. **你能接受的波动范围是多少？**
   - 例如：一年内账户最多能接受下跌百分之多少？10%？30%？50%？
   - 为什么是这个数字？（与你的收入稳定性、年龄、家庭责任有关吗？）

3. **你投资的长期目标是什么？**
   - 量化目标更佳。例如：10 年后积累 100 万投资本金、50 岁实现财务自由等。

4. **你目前最大的认知误区是什么？**
   - 例如："稳赚不赔"、"短线暴富"、"不懂股票所以不投资"等。

### 二、个人财务自检表

请如实填写以下数据：

| 项目 | 金额/比例 |
|---|---:|
| 月收入 | ____ 元 |
| 月支出 | ____ 元 |
| 月结余率 | ____ % |
| 流动资产（现金/货币基金） | ____ 元 |
| 投资资产（股票/基金/债券等） | ____ 元 |
| 负债总额 | ____ 元 |
| 应急资金可覆盖月数 | ____ 个月 |
| 每月可用于投资的金额 | ____ 元 |

计算提示：

- 月结余率 = (月收入 - 月支出) / 月收入 × 100%
- 应急资金覆盖月数 = 流动资产 / 月支出
- 每月可投资金额 ≤ 月结余（需先确保应急资金充足）

### 三、30 天行动计划

请写出一个**具体、可执行**的行动：

- 行动内容：
- 预期完成时间：
- 为什么选这个行动：
- 如何验证已完成：

示例：

> 行动内容：开通一个指数基金账户，完成风险测评，存入 3000 元试用金。  
> 预期完成时间：7 天内。  
> 为什么选这个行动：我需要先把账户体系跑通，金额小、风险可控。  
> 如何验证已完成：截图账户开立成功页面（隐去敏感信息）。

## 评分标准

AI 点评将从以下 6 个维度打分（满分 100）：

| 维度 | 权重 | 说明 |
|---|---|---|
| 完整性 | 20% | 三个部分是否都完成 |
| 逻辑清晰度 | 25% | 观点是否自洽，论证是否有依据 |
| 自我认知深度 | 25% | 是否真实面对自己的财务状况和风险偏好 |
| 数据真实性 | 15% | 财务数据是否合理、不自相矛盾 |
| 行动计划可行性 | 15% | 行动是否具体、可执行、风险可控 |
