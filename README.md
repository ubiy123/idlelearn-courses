# IdleLearn Courses

IdleLearn 智学谷的课程内容仓库。所有内容采用 **CC-BY-SA 4.0** 授权。

## 目录结构

```
investment/    投资学习路径
  L1/ 投资世界观
    01-why-invest/      # L1-01 为什么投资
      meta.json         # 课时元数据
      lesson.md         # 课程内容
      quiz.json         # 课后评测
      project.md        # 项目作业
finance/       理财基础
writing/       小说写作
templates/     项目模板与 rubric
```

## 课程单元格式

每个课时包含：

- `meta.json`：课时元数据（ID、标题、学习目标、时长、先修条件等）
- `lesson.md`：完整课程内容（学习目标、核心概念、案例分析、即时练习、延伸阅读、讨论问题）
- `quiz.json`：课后评测题（单选、多选、判断，含解析）
- `project.md`：项目/开放性作业（AI 点评 rubric 包含在内）
